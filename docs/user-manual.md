# คู่มือการใช้งานแอปพลิเคชัน Mobile BMR

## 1. การเข้าสู่ระบบ (Login)
1.  เปิดแอปพลิเคชัน **BMR Planogram**
2.  ระบุ **Branch Code** (รหัสสาขา) ของท่าน
3.  ระบุ **Password** (รหัสผ่าน)
4.  กดปุ่ม **"เข้าสู่ระบบ"**
    *   หากข้อมูลถูกต้อง ระบบจะพาไปยังหน้าหลัก (Home)
    *   หากเข้าสู่ระบบครั้งแรก ระบบอาจขอให้ท่านอนุญาตการเข้าถึงกล้องถ่ายรูป

## 2. หน้าหลัก (Home Screen)
หน้าหลักรวบรวมฟีเจอร์สำคัญ แบ่งเป็นหมวดหมู่:
*   **สแกนบาร์โค้ด:** สำหรับค้นหาสินค้าอย่างรวดเร็ว
*   **ผังชั้นวางสินค้า (Planogram):** ดูแผนผังปัจจุบันและขอปรับเปลี่ยน
*   **ลงทะเบียนสินค้า (ใหม่):** เพิ่มสินค้าใหม่เข้าสาขา
*   **ประวัติคำขอ:** ติดตามสถานะคำขอที่ส่งไปแล้ว
*   **ข้อมูลสาขา:** ดูข้อมูลสรุปพื้นที่และจำนวน Shelf

## 3. การใช้งาน: สแกนบาร์โค้ด (Barcode Scanner)
ใช้สำหรับตรวจสอบข้อมูลสินค้าว่ามีอยู่ใน Planogram หรือไม่
1.  กดเมนู **"สแกนบาร์โค้ด"**
2.  นำกล้องส่องไปที่บาร์โค้ดสินค้า
    *   **หากพบสินค้าใน Planogram:** จะแสดงเครื่องหมายถูก ✅ และบอกตำแหน่ง (ชั้น/ตำแหน่ง)
    *   **หากไม่พบ:** จะแจ้งเตือน ❌ ว่าไม่พบสินค้าในระบบ
3.  ท่านสามารถกด **"พิมพ์บาร์โค้ด"** ด้านล่าง หากสแกนไม่ติด

## 4. การใช้งาน: ดูและปรับเปลี่ยน Planogram
1.  กดเมนู **"ผังชั้นวางสินค้า"**
2.  เลือก **ชั้นวาง (shelf)** ที่ต้องการดู
3.  ระบบจะแสดงรายการสินค้าในชั้นนั้น
    *   สามารถกดที่สินค้าเพื่อดูรายละเอียด
    *   สามารถกดปุ่ม **Edit** (ถ้ามีสิทธิ์) เพื่อขอ ย้าย/ลบ/แก้ไข สินค้า

## 5. การใช้งาน: ลงทะเบียนสินค้า (Register Product)
ใช้สำหรับเพิ่มสินค้าใหม่ที่ยังไม่มีในระบบ Planogram ของสาขา
1.  กดปุ่มสีเขียว **"ลงทะเบียนสินค้า"** ในหน้าหลัก
2.  **สแกนบาร์โค้ด** สินค้าที่ต้องการเพิ่ม
    *   หรือกด "พิมพ์บาร์โค้ด" เพื่อกรอกเอง
3.  ระบบจะตรวจสอบ:
    *   ถ้ามีสินค้าใน Master Data แต่ยังไม่อยู่ในสาขา -> **ไปขั้นตอนถัดไป**
    *   ถ้ามีสินค้านี้อยู่แล้ว -> ระบบจะแจ้งเตือน ⚠️ และบอกตำแหน่งเดิม
4.  **เลือกตำแหน่งจัดเก็บ:**
    *   เลือก **ชั้นวาง (Shelf)**
    *   เลือก **ชั้น (Row)**
5.  ระบบจะคำนวณตำแหน่ง (Index) ต่อท้ายให้อัตโนมัติ
6.  กด **"ยืนยันลงทะเบียน"**
7.  เมื่อสำเร็จ หน้าจอจะแสดง ✅ และท่านสามารถสแกนสินค้าชิ้นต่อไปได้ทันที

## 6. การดูประวัติคำขอ (Request History)
1.  กดเมนู **"ประวัติคำขอสาขา"**
2.  ท่านจะเห็นรายการคำขอที่ส่งไปล่าสุด
3.  **สถานะ:**
    *   🟠 **Pending:** รออนุมัติ
    *   🟢 **Approved:** อนุมัติแล้ว
    *   🔴 **Rejected:** ถูกปฏิเสธ (พร้อมเหตุผล)

## 7. ปัญหาที่พบบ่อย (Troubleshooting)
*   **สแกนไม่ติด:** ลองเพิ่มแสงสว่าง หรือกด "พิมพ์บาร์โค้ด" เพื่อกรอกเลขเอง
*   **เข้าใช้งานไม่ได้:** ตรวจสอบรหัสสาขาและรหัสผ่าน หรือตรวจสอบการเชื่อมต่ออินเทอร์เน็ต
*   **ไม่พบชั้นที่ต้องการ:** ติดต่อ Admin เพื่อตรวจสอบการตั้งค่าพื้นที่สาขา
